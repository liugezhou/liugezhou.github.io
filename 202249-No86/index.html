本周主题:积极心态

心态是一个人成长、成功与失败的关键因素，它能够影响我们的心情、态度与行为。
所以，积极心态狠是重要的，它能够让我们获得更多的乐趣、快乐与成功。

积极心态的重要性，首先体现在阅读上。
阅读正能量、温馨的文章，能够给我们带来积极的心态，让我们看到人生的美好，感受到人间的温馨。
比如，阅读一篇感人至深的小说，能够让我们感受到爱的力量；
阅读一篇幽默风趣的故事，能够让我们抛开忧虑，释放心情，能够让我们获得快乐与愉悦。

此外，积极心态还能够让我们与周围的人相处得更好。
正能量、温馨的文章能够给读的人带来心情的愉悦与感动，让我们的心与周围的人产生共鸣。 比如，我们与朋友聊天时，能够给对方带来安慰与鼓励；我们与同事相处时，能够给对方带来支持与鼓舞。所以，积极心态能够让我们与周围的人相处得更加融洽。

然而，悲观一类的文章才是引起共鸣、吸引眼球的流量密码，除了增加局外人的愤怒、增加似乎拥有的家国情怀外、体现的不过是“徒有虚表”的那种满怀苍生般意淫的心灵拯救。
当看到过多的坏事情在身边徘徊，缺少了积极心态，就会把自己的幸福忽略掉，甚至产生悲观、消极情绪，得不偿失。

so，请保持积极的心态。

个人信息流分享🔗

简单总计一下作者的这篇个人信息流的阅读记录流程。

  1. 输入，即订阅的信息源。
    作者通过订阅公众号推送、B站Up主、V2ex、少数派、掘金、豆瓣小组、知乎热榜、RSS、newsletter等，将所有数据统一输入到「inoreader」上。
  2. 采集，通过自动化初筛信息源。 通过「inoreader」的自定义规则监控，通过Webhook转发服务(inoreader需付费)。
  3. 阅读文章并标注笔记 使用 Reeder 或者 inoreader 阅读均可。
    使用类Vim键快速阅读，标注划线即可。
  4. 整理输出 使用Logseq或者Obsidian定期同步标注笔记到本地笔记中，之后进行整理加工。

PS:「inoreader」的真是非常赞啊，对RSS等的强大支持，体验了一个小时就有忍不住购买会员的冲动。

无论如何我们都处于一个伟大的时代，原因不为别的，而是因为AI已经突破图灵测试，未来的发展无法想象。从谷歌机器学习论文发布到现在，科幻小说里面的事情一件件蹦出成为现实。这明明是一个AI帝国的前夕，我们就别为人类旧时代的阴暗而太伤神了。 Michael Anti
