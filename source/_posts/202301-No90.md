<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">不知道其它的周报写作者是一个什么状态。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">对于最近的我来说，</p>
<ul data-tool="mdnice编辑器" style="margin-top: 8px; margin-bottom: 8px; padding-left: 25px; font-size: 15px; color: #595959; list-style-type: circle;">
<li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">一会看到一个工具想简单记录一下，找到赞美它的点。</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">一会看到一本书，还没开始读，就已经在想写心得了。</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">一会看到一个开源项目，赞叹它的优雅之外，又想自己折腾一番介绍使用、部署为己所用。</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">一会看到一句鸡汤，又想记录下来准备meditation一下。</section></li></ul>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">倘若我做的头头是道也就不会在这逼逼咧咧，由于todolist太多，造成的结果是都想做、都不理想。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">我回想最开始写周报的样子：<br>
开始的时候遇到一些好玩的工具、新闻、文摘、图片后觉得很赞，而我又不太善于基于社交的分享(多数的人或许也有同样的问题吧)，可人需要社交，这就造成我势必会想找个人说道说道。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">信息流爆炸，这个工具还没赞美出去，那边又来了另一个冲刷大脑的八卦。<br>
久而久之，有太多的东西、想法需要去输出了，当初感觉哪怕有个小黄鸭，我也要说一遍，不然这个信息似乎就是没有消化。<br>
这一个一个转移注意力的网页让人应接不暇，稍纵即逝，最后想到最能够沉淀的方式就是记录。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">最开始的定位就是写给自己看，记录备份好玩的东西算是一个 个人信息字典，也就几乎不考虑内容的质量、关注、阅读量等等了。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">后来慢慢收到了一丁点的关注后，即使知道最忠实的读者还是自己，也还是发生了一丝变化：<br>
现在对于文章在各个平台的访问量、关注数、留言等，相比文章本身而言更加分神，也就有了开头提到的一会这个，一会那个。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;"><strong style="color: #3594F7; font-weight: bold;"><span>「</span>记录更应该的是忠于自己，而不是娱乐别人<span>」</span></strong>，一篇文章的记录若是脱离了真挚的想法输出，为什么而什么，他是没有灵魂的，他应该回归他本来的样子。</p>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">一些事情最开始决定的时候往往就是开始的样子了，对于我来说，比如：</p>
<ul data-tool="mdnice编辑器" style="margin-top: 8px; margin-bottom: 8px; padding-left: 25px; font-size: 15px; color: #595959; list-style-type: circle;">
<li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">大学刚入学发放的银行卡初始密码，后来成为了所有的密码项</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">对于一瞬间随便想到的昵称，后来就成为了所有社交平台的昵称。</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">对于曾经碎碎念念的一些小事，过去了好多年还是在那几个方面叨逼叨</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">对于开始随机拟定的一个想法，慢慢也就成为了后来的输出原则</section></li><li><section style="margin-top: 5px; margin-bottom: 5px; line-height: 26px; text-align: left; font-size: 14px; font-weight: normal; color: #595959;">对于新的想法，你以为你是在寻找他，其实一直在深化他，辅佐证明他，不可改变</section></li></ul>
<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">回归到他最开始的样子，然后直接探索到要达到的样子，是什么，就随它去吧。</p>

<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">"37%法则"的意思就是，寻找阶段进行到37%就要停止。</p>

<p data-tool="mdnice编辑器" style="padding-top: 8px; padding-bottom: 8px; line-height: 26px; color: #2b2b2b; margin: 10px 0px; letter-spacing: 2px; font-size: 14px; word-spacing: 2px;">首先抛出了两个问题：<br>
