<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">上周在Github上看到一个Vim教程的仓库,跟着教程学习,在博客上发了一篇技术文：<a href="https://blog.liugezhou.online/030-%E4%B8%80%E7%AF%87%E5%B0%B1%E5%AD%A6%E4%BC%9Avim" style="word-wrap: break-word; font-weight: bold; color: rgb(60, 112, 198); text-decoration: none; border-bottom: 1px solid rgb(60, 112, 198);">《一篇就学会vim》</a>。<br>

<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">和平常比很不平常，通过百度统计看到大多数的访问来源是weibo。<br>
于是去weibo中搜相关资讯，果然发现被一互联网科技博主推荐。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我就想了一下，为什么可以收获这么多的曝光。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">第一点毋庸置疑的是流量推荐。<br>

第三点是因为这篇文章的标题有靴微的诱惑力： <code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">一篇就会</code>。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">速成的渴望在心间徘徊，想到我收藏了很多的课程，许多绚丽的网站，书单、电影，列出了很多的TODO List。信息的爆炸、专注力的持续下降，许多一直躺在收藏夹吃亏。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">记录于此，就是提醒自己，把收藏夹拿出来晾晾，找找想做的事，读读想读的书，多尝试几次，总会有一件宝贝，让人爱不释手。</p>
