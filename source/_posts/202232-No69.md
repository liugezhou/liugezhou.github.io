<h1 data-tool="mdnice编辑器" style="padding: 0px; font-weight: bold; color: black; font-size: 24px; text-align: center; background-image: url(https://cdn.staticaly.com/gh/liugezhou/image@master/blog/blog_title2.jpg); background-position: center top; background-repeat: no-repeat; background-size: 95px; line-height: 95px; margin-top: 38px; margin-bottom: 10px;"><span class="prefix" style="display: none;"></span><span class="content" style="font-size: 20px; color: rgb(60, 112, 198); border-bottom: 2px solid #3C7076;">本周主题:我的前半生</span><span class="suffix"></span></h1>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">标题起的有点大，算是一个回忆贴，贴一下即将步入三十岁的老男人，之前体验过的一些兼职，是对过去一个个短暂体验后的微记录。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">不去回忆真是不记得干过什么。<br>
整体上来说，体验过销售、电工、快递员、作家、运营、保险、自媒体、个体户、保险、文艺 等。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">时间紧迫，下面回忆在一气呵成后，未进行二次编辑。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">电工</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">当电工的经历是在大学暑假，某一铁矿下井。<br>
主要从事水泵员和电工两个专业。<br>
主要工作内容就是看水坝、换灯泡。<br>
上十二休息二十四，戴安全帽、穿靴子、厚厚的工作服，乘坐矿罐下井大概几百米不记得了，矿下无论春夏秋冬都非常凉爽。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">矿下经历的记忆点就是：看水库水满后要及时放水、各个黑不隆咚的地方需要接线换灯泡、印象最深的莫过于矿井下放炮钻眼：霹雳吧啦--雷声根本啥也不算。<br>
一天收入90，干了一个月，收获2700.</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">快递员</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大学那会，闲的没事，去圆通快递送了一个月的快递。<br>
跟着一退伍的年轻师傅送快递、寄快递。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">每天的工作内容就是，早晨起来分片区分件、一件一件快递在地上扔来摔去，热火朝天。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">快件分好后，骑上三轮车，顺着一条街开始挨家挨户上门送货、打电话、敲门、送件。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">到中午的时候吃完饭，接着回圆通点再等一波货车、再次取件。<br>
下午的量没那么多，四点左右送完，然后去一些淘宝等需要发货的商家家里(居民楼)，上门取件，印象深刻的就是：快递件在仓库横飞、快递费由快递员说了算、淘宝商家很多都是自家包装。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">入职的时候心理预期是干一个月，但面试的时候说是我要长期从事这个行业。<br>
干完一个月的时候，提出离职，由于很儿戏，老板扣了一些钱，貌似给发了一千一左右吧。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">保险</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">体验过保险，是仅仅限于培训这个经历。<br>
彼时还在写代码，大学一哥们从事保险工作，为了完业绩，介绍我去酒店听讲座。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">在一顿花言巧语下，在免交钱的情况下，酒店进行了两夜三天的保险讲座。<br>
具体讲了什么内容不记得了，只是印象有些蛊惑，每天穿着小西服、唱着美丽的赞歌、恨不能马上从事这个职业。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">三天后新鲜劲一过，对各种保险的工作内容做了个了解，然后就没然后了。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">销售</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大学的时候，卖健力宝。<br>
在石家庄宫家庄一块的小吃一条街，貌似去了两周、一天四十左右？记不太清了。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">个体户</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">也是大学的时候，在石家庄老车站旁边，进了一批钱包，大概进了差不多200的货？</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">进货价从8块到13块不等，卖20-30。<br>
在夜市、校门口摆过地摊，再加上朋友支持，估摸着卖了一半。<br>
总体没亏，留下的一半好似都送人了。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">印象还是觉得脸皮太薄。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">文艺工作-Dancer</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">也是大学的时候，痴迷跳舞。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">后来认识一哥们，是正定某街舞培训的老师。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大学没事总是逃课出去玩、演出、比赛。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">记忆深刻的是接过婚礼邀请表演，费用多少不记得了。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">后来因为一次正定金河舞蹈大赛，摔了膝盖，这门热爱的运动就此夭折。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">现在想起那会傻乎劲，如若没有伤膝，搞不好现在是一名街舞老师。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">天意不可违啊。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">文艺工作-出版</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">也算出版行业吧，Wakaka。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">也是大学期间的某个暑假，来北京找朋友玩。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">在马驹桥一带找了个兼职，找到一个压身份证、进工厂、捆书的兼职。<br>
妥妥的出版行业，捆的书貌似是教科书。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">收入100左右。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">文艺工作-作家</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大概2021年，起点发表了一个小说，总计四章不足一万字。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">字字瞎J扯，感觉还不错，但没坚持下去。<br>
看到起点个人主页的作家称号、让人感觉甚是🐮🍺轰轰。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">收入为0，感受是缺乏天马行空的想象。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">炒股</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大概2020年那会，蒙眼入场，佛系炒股。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">年底的时候赚了两万多。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">遂觉股神下场，21年亏了不到两万，深埋至此，扒不出来，软件卸载、眼不见、心不烦。<br>
就在刚刚，眼睛一瞥，三分之一买入价已跌去，惨不忍睹，等待回光返照。</p>
<h4 data-tool="mdnice编辑器" style="margin-top: 30px; margin-bottom: 15px; padding: 0px; font-weight: bold; color: black; font-size: 18px;"><span class="prefix" style="display: none;"></span><span class="content">运营</span><span class="suffix" style="display: none;"></span></h4>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">运营是在做程序员后端开发过程中的，其中工作的一部分。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">当时进公司是是面试开发的，然后总是问关于对销售的想法，以及今后出国去新加坡卖软件的看法。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">没得选，面试的时候，点头哈腰、说啥都行，只求给份工作。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">于是，为了把我往销售方向上带， 紧张的996的开发同时，给了一些运营号(QQ、Wechat、脸书)，每天的工作内容就是加好友。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">头像清一色美坯，性别一律为女，QQ当时加了多少人不记得了，Wechat加了差不多不到两千人。 脸书也因为某些原因给🈲️掉了。</p>

<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">印象中有过收入的是两个平台</p>

<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">然后就是一些小小记录</p>
