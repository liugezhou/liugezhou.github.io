<h1 data-tool="mdnice编辑器" style="padding: 0px; font-weight: bold; color: black; font-size: 24px; text-align: center; background-image: url(https://cdn.staticaly.com/gh/liugezhou/image@master/blog/blog_title2.jpg); background-position: center top; background-repeat: no-repeat; background-size: 95px; line-height: 95px; margin-top: 38px; margin-bottom: 10px;"><span class="prefix" style="display: none;"></span><span class="content" style="font-size: 20px; color: rgb(60, 112, 198); border-bottom: 2px solid #3C7076;">本周主题:假期结束</span><span class="suffix"></span></h1>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">每到这个时候，我就像那些时刻在关注社会新闻的媒体人一样：焦虑的寻找社会热点、搜刮小作文素材，这一周的主题记录些什么？</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">虽然这个过程算不上痛苦，但确实很是犯难，每周的主题总是放在最后，半天憋不出来东西。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">没有笔墨非要抒发显得很做作。可<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">本周主题</code>这个模块的目的就是为了搜刮想法，进行一些思考，记录一些想法，养成做作的习惯。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">写了这么几期后，菜越发的发现根本就没有自己的想法。倘若没追个热点，没看些观点，没看些评论，根本就产生不了自己的想法。一次一次的验证想法后，自以为对的东西会愈加坚固，可坚持认为对的一定是对的吗？<br>
那肯定不是。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">能确定对的是无数事件证明权力一定是应该关在笼子里的。<br>
<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">贵族重权利，百姓重礼仪</code>。<br>
突然觉得：在走向共同富裕的道路上，又不打击社会精英积极性，真是一门学问啊。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">最后再突然觉得一下：那些敢于批判、敢于说出真话的人，某些方面看真是一个不同寻常的人。<br>
似乎仅仅只是一个不同寻常的人了。</p>

<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">凑个热闹，总结一下2022年诺贝尓物理学奖到底研究了个什么。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">首先，2022年的诺贝尔物理学奖颁发给了在<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">量子力学</code>领域有突出贡献的三位科学家，他们分别来自法国、美国和奥地利。
奖励他们用纠缠光子验证了量子不遵循贝尔不等式，开创了量子信息学。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">然后，什么是量子纠缠？贝尔不等式又是什么呢？</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">首先，经典物理学或者说传统物理学，或者说爱因斯坦的相对论，说的是任何东西的传输都是有时间的，这个时间的上限是光速。<br>
量子力学不遵循这个规律，比如这个<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">量子纠缠</code>,一对纠缠的光子对，只要测量了其中一个光子的状态，另外一个光子不用测量也能知道它的状态。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">之前我在看 <a href="https://mp.weixin.qq.com/s/-u3b1E9UXbO5H8attOeNvw" style="word-wrap: break-word; font-weight: bold; color: rgb(60, 112, 198); text-decoration: none; border-bottom: 1px solid rgb(60, 112, 198);">七堂物理极简课</a> 这本书的时候，总结如下：</p>
<figure data-tool="mdnice编辑器" style="margin: 0; margin-top: 10px; margin-bottom: 10px; display: flex; flex-direction: column; justify-content: center; align-items: center;"><img src="https://cdn.staticaly.com/gh/liugezhou/image@master/blog/202240_02.webp" alt style="display: block; margin: 0 auto; max-width: 100%; border-radius: 4px; margin-bottom: 25px;"></figure>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">可以简单理解为他们的获奖证明了爱因斯坦的错误，光速这个限制在量子力学中不成立。</p>
