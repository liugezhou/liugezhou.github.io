<h1 data-tool="mdnice编辑器" style="padding: 0px; font-weight: bold; color: black; font-size: 24px; text-align: center; background-image: url(https://cdn.statically.io/gh/liugezhou/image@master/blog/blog_title2.jpg); background-position: center top; background-repeat: no-repeat; background-size: 95px; line-height: 95px; margin-top: 38px; margin-bottom: 10px;"><span class="prefix" style="display: none;"></span><span class="content" style="font-size: 20px; color: rgb(60, 112, 198); border-bottom: 2px solid #3C7076;">本周主题:为什么要打架</span><span class="suffix"></span></h1>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">鉴于最近气氛烘托的比较热火，我就杜撰了一个故事，也算是记录本周大事记吧。<br>
这个故事最开始的时候，我是用来解释为什么这个<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">架</code>要打，然后到后来又引发了一些联想。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我是男主角中仔，我的妻子是湾妮，我们是一家人，然后由于一些history的原因我们吵架，还动过手，因此不是很和睦，还在互相闹脾气，但毕竟还是睡在一个屋子里的合法夫妻。<br>
可突然一个美男子的健身摇滚歌手，竟就趁虚而入，我的妻子<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">或许是为了气我</code>、或是一些<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">我可能知道的原因</code>，对美男子也是暗送秋波。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我虽然很愤怒，但为了自身发展，我也在努力的学习美男子的健身技巧、学唱歌，并且跟美男子还有一些生意上的往来，也就一直隐忍，对此事闷在心里。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">可突然有一天，美男子和妻子竟当众约好来我家的阳台品酒，我自然是极力阻止的，这是我家，我也是个男人，我还能不知道你们男人的花花肠吗？你说你们背后搞小动作也就算了，还竟敢约在我的家里阳台上，我能忍的了？</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">那我肯定得有一番准备，敢进家我就得揍你吧，这是赤裸裸的挑衅啊。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">就基于这个想法，我觉得怎么也得打一顿啊。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">做了这么多心理建设，我还发了好几个微博，没想到最后美男子还是恬不知耻的还是进家来了。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">大家也知道了，我目前的做法是在家里打开了电视机，按上了麦克风，放大了声音来表达我的抗议。<br>
当然这是由我内心最理智的声音决定的，毕竟打架的危害很是不可估量的，百害无一利。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">声音响着，我的脑子也在转着，是什么导致今天这步田地，一定是多方面的，我也为此，从某一个方面自省了一下：</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我自身是一个权威极重，一意孤行、不愿听任自己的细胞抗议等反对声音的（是有那么一点老顽固的意思）。但我还是觉得我一切是为了大业，只能对自己的细胞采取无视态度(指不定哪个细胞犯上作乱呢)。<br>
我一心发展为往前冲只为健身，不说其它技艺，起码得赶紧追上美男子矫健的肌肉啊，所以一定的牺牲是必要的。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">然后一些<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">感性</code>的细胞告诉我：</p>
<blockquote class="multiquote-1" data-tool="mdnice编辑器" style="border: none; font-size: 0.9em; overflow: auto; overflow-scrolling: touch; background: rgba(0, 0, 0, 0.05); color: #6a737d; padding-top: 10px; padding-bottom: 10px; padding-left: 20px; padding-right: 10px; margin-bottom: 20px; margin-top: 20px; padding: 15px 20px; line-height: 27px; background-color: rgb(239, 239, 239); border-left: none; display: block;">
<p style="padding-bottom: 8px; padding-top: 23px; margin: 0px; line-height: 26px; padding: 0px; font-size: 15px; color: rgb(89,89,89);">你知道你不是非常的爱惜自己细胞吗？有时候脾气还很臭、特有的专横也算是出了名的，我们知道你也想改，但是你就是我，颜色不一样的烟火，即使如此，我们也还是爱你的啊。</p>
</blockquote>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我的湾妮呢，她温柔、独立，并且平时非常注重饮食、相对我来说，她对自己的细胞要友好那么一点（具体这一点是多少，我也不清楚），她如今的抉择，一方面确实是由于她不顾道德法则只为自己欢乐，另一方面可能也是她的细胞在作祟吧。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">没有条理的碎碎念了一阵后，我我我到底该怎么办呢？</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我想我能做的也就是，在努力健身、拥有矫健肌肉的同时，我也应该在饮食方面上有一点注意，对自己的细胞多一些关怀，也培养一下摇滚的兴趣，不能总是那么严肃，嗯，对，不能那么严苛，对自己细胞的管治是好的，但不能一刀切，这个趋势越来越明显，这样不好。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">我还是得增加一些幽默细胞，并且我也要多多培养自己的爱好，美男子的那些有的我都要有，毕竟好的东西是令人向往的，是积极的，那些不好的东西，他<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">抽烟喝酒</code>我就不要了，并且我还有一些他没有的，等着他也来向我学习、向我看齐的时候，想必湾妮肯定会回心转意吧。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">不管怎么样，总结就是：家不能散，我能做的就是继续健身、更重要的是关注一些我细胞的感受、多一些幽默、多一些创意、摇滚的、民族的、说唱的、流行的该适度放开了让他们自由发展，我还得学学画画，还得弹弹钢琴，我那么多亿的细胞，我得发挥主观能动性，我不能对他们如此的限制了，我知道我还有诸多的不足，但是只要我不那么专横，听取更多细胞的声音，我想我肯定会越来越好的。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">毕竟当初我去健身的时候，也是很多人反对的，可现在不是越来越好吗。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">嗯，这么想想是挺好的。</p>
