<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;"><code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">享受</code>100期周报、<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">习惯性</code>100天前端、<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">沉浸式</code>100本读书笔记。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">前几天还想着展开论述一下<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">享受</code>做一件事带来的感受积极性的时候。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">这周日去天安门逛了一逛，没收了一个打火机，里里外外转了好几圈，步行好几步路路，晚上到家，困意袭来，顿时蔫了，什么积极性的不要了。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">还是坚持实实在在。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">那话怎么说来着，不是因为什么什么才坚持 ，是因为坚持才什么什么。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">目前也还不知道什么什么是什么，但总归是个什么。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">坚持什么不重要，主要是通过种种表现内涵自我，将有限的时间浪费在了看得见的事情之上，从中汲取了一些营养，留下了淡淡字迹。</p>
<p data-tool="mdnice编辑器" style="padding-bottom: 8px; margin: 0; padding-top: 23px; color: rgb(74,74,74); line-height: 1.75em; font-size: 17px;">内容是什么不重要，<code style="font-size: 14px; word-wrap: break-word; padding: 2px 4px; border-radius: 4px; margin: 0 2px; background-color: rgba(27,31,35,.05); font-family: Operator Mono, Consolas, Monaco, Menlo, monospace; word-break: break-all; color: rgb(60, 112, 198);">意识形态</code>告诉我们，不管内在如何，<u>表象一定要顺理成章</u>。</p>
