2-4 介绍GraphQL的使用和特点

为什么选择Restful API，而没有选择GraphQL

  • GraphQL(Graph Query Language) 图查询语言。
  • 意思是擅长处理"图"数据结构的查询，即多个数据对象，各个之间还有关联关系。
  • 核心概念：schema rootValue

2-5 选择Restful API 而非 GraphQL

应用场景

  • 数据关系比较复杂
  • 前端查询需求多变
  • 有一个独立的数据提供方，对接很多使用方，不能一一定制开发
