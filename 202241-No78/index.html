本周主题:活着

《 活着》是中国当代作家 余华创作 的长篇小说，首次发表于《收获》1992年第6期。

在整理这篇文章的时候，去猫了一眼电影的最后结尾，大相径庭。

阅读背景

这本书最初的时候是在好几年前的一个除夕的夜晚，和两个高中好友广场游荡。

他们同时提到了这本在读的书，并交换着心得，我没看过，谈论起来只是嗯嗯称道，听他们讲述，当时并没有什么体会。

而这次机缘巧合读起来这本书,瞬时的源头是在看到这张梗图后：

我是在通勤的上下班的地铁上、腿着看完的。

一边读一边叹，文字叙述朴素、讲述平淡。像我如此铁石心肠之人，竟也禁不住几次眼眶湿润，这究竟是何等的魔力，何种剧情，能用文字调起人如此的情绪感受。

叙述没有那么刻意、剧情推进也是缓和有序，可串起来后一个又一个的打击真是太悲了。
看完的时候，除了感受文笔力量之强大之外，面对平淡无奇的生活，唯知足尔。

后来我就忍不住跟我的老伴分享，也没忍住透露了内容，实在该死，怎么能剧透呢。

老伴听我评价如此之高，啧啧称奇，于是也就在上下班的通勤路上看完了这本书。

也就泪眼婆娑。

随之心潮澎湃的写了一篇她的读后感，这个链接是她的原文内容。

文摘

不要强迫自己做任何内心讨厌的事情。因为一旦你习惯了做这些事情，就很难下决心退出了。

很长一段时间，我真正的生活看似马上就要开始了，但是总有一些阻碍阻拦着，有些事得先解决，有些工作待完成，时间貌似还得花，还有一笔债务要去付清，然后生活就会开始。最后我终于明白，这些障碍，正是我的生活。这么想让我意识到，没有通往幸福之路，幸福就是路途本身。所以，珍惜你拥有的每一刻，记住时间不等人。

maigret

maigret，是一个开源的用户名查找工具，目前支持全球3000多个站点的数据。
这个工具是一个python库，可以从3000多个网站匹配查找存在的用户社交账户资料页，国内外的网站都有，搜索结果返回符合的用户名的社交页面资料链接，用户名不支持中文（大概是数据库不好存的缘故）。

Stormah: Own Your Data

一个简易的网页笔记，可以指定远程 Git 仓库作为后端。你在浏览器里做笔记，储存到远程的 Git 仓库。
先码住。

NoSleep.js

这个 JS 库可以让手机不进入休眠状态，网页窗口始终亮屏，原理是在页面插入一段不可见的、一直在播放的视频，代价是增加 CPU 负载和耗电。
使用步骤：

1. npm install nosleep.js
2. import NoSleep from 'nosleep.js'
3. var noSleep = new NoSleep()
4. // Enable wake lock.
// (must be wrapped in a user input event handler e.g. a mouse or touch handler)
document.addEventListener('click', function enableNoSleep() {
  document.removeEventListener('click', enableNoSleep, false);
  noSleep.enable();
}, false);

// Disable wake lock at some point in the future.
// (does not need to be wrapped in any user input event handler)
noSleep.disable();
